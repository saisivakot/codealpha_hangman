# 🎮 Hangman Game in Python

A simple command-line Hangman Game developed using Python.

## 📌 Project Description

This project is a classic Hangman game where the player guesses a randomly selected word one letter at a time. The player has six chances to guess incorrect letters before the game ends.

---

## 🚀 Features

- Random word selection
- Input validation
- Prevents duplicate guesses
- Displays remaining chances
- Win and Game Over messages
- Beginner-friendly Python project

---

## 🛠 Technologies Used

- Python 3
- Random Module

## ▶️ How to Run

1. Install Python.
2. Open Command Prompt.
3. Navigate to the project folder.
4. Run:

```bash
python hangman.py
```
